### Client <a name="client"></a>
Clone the repository using the following command :
```bash
git clone https://github.com/OpenQuantumDesign/quantumion
```
Install with pip :
```bash
pip install .
```

### Server <a name="server"></a>

#### Docker Compose <a name="docker-compose"></a>
```bash
cd docker
docker compose up -d
```
